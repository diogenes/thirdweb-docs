# Instalando um Contrato Pré-construído

[Contratos Pré-construídos](/pre-built-contracts) são a forma mais rápida de começar a construir.

Está página irá mostrar para você como instalar o contrato pré-construído [NFT Drop](/pre-built-contracts/nft-drop) na rede de teste Goerli.

Clique no botão abaixo para ir até o Dashboard, onde você poderá criar e instalar seu contrato inteligente.

## Conectando sua carteira

Se esta é sua primeira vez desenvolvendo para a Web3, you precisará de uma **carteira** para interagir com a blockchain.

Você também pode obter gratuitamente **fundos de teste** à partir de um **faucet** para pagar pelas gas fees de suas transações.

## Criando o seu contrato

Antes de você instalar, você pode configurar alguns metadados do seu contrato, como o nome, descrição e a imagem!

![contract metadata](./assets/configure-contract-metadata-trimmed.png)

Abaixo dos metadados do seu contrato, selecione a rede de teste `Goerli (GOR)`, então clique em `Deploy Now`!

![Select Goerli Test Network](./assets/select-goerli.png)
