# Visão Geral da Plataforma

<div className="heroGrid">

<div>
<p>thirdweb é um framework de desenvolvimento que habilita você a construir funcionalidades da web3 para suas aplicações.</p>

<p>Nós fornecemos fluxos que aceleram seu desenvolvimento, incluindo:</p>

1. **Contratos**: você pode usá-los para construir a fundação das funcionalidades web3 de suas aplicações.
2. **SDKs**: use eles para criar aplicações que interagem com a blockchain nas suas linguagens favoritas.
3. **Dashboards**: usados para gerenciar as configurações dos seus contratos inteligentes, permissões dos membros do seu time, fontes de receita e dados analíticos.

## Contratos

<div className="heroGrid">

<div>
  <p>Instalar contratos inteligentes é o primeiro passo para construir qualquer aplicação web3.</p>

Você pode instalar um dos nossos [contratos pré-configurados](/pre-built-contracts) para os casos-de-uso mais comuns (NFTs, tokens, marketplaces),
ou você pode construir seu próprio contrato inteligente com nossas [Extensões](/extensions) e disponibiliza-lo com o [Deploy](/deploy).

## Dashboard

<div className="heroGrid">
  
  <div>
    <p>
      Fornecemos à você um intuitivo <a href='https://thirdweb.com/dashboard'>Dashboard</a> para gerenciar suas aplicações.
    </p>

Nele você pode realizar operações comuns para administrar seus projetos sem qualquer código, como o seguinte:

- **Instalando** novos contratos inteligentes e **interagindo** com os contratos atuais
- **Configurando** _royalties_ e taxas da plataforma ou outras configurações do contrato
- Emitindo **permissões** a nível de time para promover a colaboração
- Coletando **informações** à partir dos dados analíticos do seu contrato inteligente

</div>
